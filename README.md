## Set Up

python venv create

```
$ python -m venv myvenv
```

start venv

```
# windows
$ . venv/Scripts/activate
```

or

```
# mac, linux
$ source venv/bin/activate
```

pip upgrade

```
(venv) $ pip install --upgrare pip
```

python module install

```
(venv) $ pip install requirements.txt -r install
```

django runserver

```
(venv) $ python manage.py runserver
```
